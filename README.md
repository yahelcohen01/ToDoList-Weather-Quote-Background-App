# ToDoList-Weather-Quote-Background-App
React Todo list app with weather(openweatherAPI), inspirational quote(adviceslip API) and Sliding Background Image(unsplashAPI).

Libraries used:
1. React JS
2. React-Redux
3. react-confetti-explosion
4. Axios
